インストール方法
---------------

後方互換性のない変更が雑に加えられる可能性があります。
ご研究で使用される場合は、バージョン指定をしたうえでインストールすることをオススメします。

.. code-block:: bash

   pip install git+https://github.com/issakuss/issych
   pip install git+https://github.com/issakuss/issych@v0.0.6.1  # ←バージョン指定する場合はこう
